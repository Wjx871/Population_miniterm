# 微信小程序移动管理端 V1.0.0 发布说明

发布日期：2026-07-15

用户可见版本：`V1.0.0`

## 版本定位

面向人口数据库管理系统内部工作人员的微信小程序 V1，聚焦移动查询、申请进度查看和审批处理，复用现有后端认证、权限和数据范围。

## 页面与功能

- 13 个正式页面：登录、工作台、业务中心、办理中心，人口/家庭户/申请/审批的列表和详情，个人中心。
- 四项原生底部导航：工作台、业务、办理、我的。
- 真实指标、脱敏查询、分页/刷新、申请下一步、材料与进度、移动审批和个人职责。

## 用户角色

支持现有 `admin`、`viewer`、`population`、`household`、`approver` 五类账号。所有入口和操作依据 `/api/auth/me` 返回的权限，后端继续执行接口权限和 ALL/DEPARTMENT/REGION 数据范围。

## 核心改进

- 统一的用户任务导向信息架构和四 Tab 导航。
- 登录、工作台、人口、家庭户、申请、审批和个人中心的最终视觉/交互优化。
- 本地 Tabler Outline 图标系统，不使用手绘、字符图标、远程字体或 CDN。
- 统一加载、空、错误、敏感文本和状态组件，并保留原始状态进行业务判断。
- 请求空参数统一清理，首次加载去重，错误重试只由用户触发。

## 已知限制

- V1 不包含公众注册/OpenID、人口和家庭户写操作、复杂申请创建以及尚未移动化的专业业务页面。
- 微信开发者工具双机型、Console/Network、真机和 13 张正式截图待用户完成。
- 真机调试需局域网或部署后端；正式发布需 HTTPS 合法域名。

## 运行环境与部署要求

- 后端运行在 Java 17，提供可访问的 HTTPS API，并保留 JWT、权限和数据范围校验。
- 将 `miniprogram` 导入微信开发者工具，个人地址仅写入被 Git 忽略的 `config/local.js`。
- 正式 AppID、AppSecret、Token、数据库密码和个人 IP 不进入仓库。发布前配置 `request/uploadFile/downloadFile` 合法域名和可信证书。

## 验收结果

小程序 123/123 测试与 13 页静态检查通过；后端 `clean test/package` 各 327 项通过；PC 前端 195 项单元测试、检查、生产构建和 20 项 Playwright E2E 通过。自动化门禁达标，可推送功能分支并创建草稿 PR；在微信开发者工具人工验收和正式截图完成前，不建议直接合并 `develop`。
