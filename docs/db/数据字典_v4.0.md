# 数据字典 v4.0（字段参考）

> 本文档对应 `sql/init.sql` 中的 39 类字典项；每条记录标注了：**编号 / 类型 / code / 含义 / 引用字段**。
> 业务方在 DB 与 Service 层都应使用 `dict_code` 而非 `dict_label`，避免翻译导致脏数据。
