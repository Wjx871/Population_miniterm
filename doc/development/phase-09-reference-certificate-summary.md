# Phase 09 行政区划、数据字典与通用证件实施总结

Phase 09 新增唯一 `admin_region` 自关联模型与五级演示层级，提供一次查询构树、直接子节点、条件查询和受控维护；参考数据查询不错误套用业务数据范围。现有 `data_dictionary` 补齐 H2 结构、version、查询和管理员维护，未引入缓存，也未强制替换既有安全状态枚举。

现有 `certificate` 增加乐观锁和软注销字段，提供分页、详情、按人员查询、创建、修改和注销。证件范围在 Mapper SQL 中关联 person 执行，响应统一脱敏；到期状态查询时计算，不新增调度任务。居住证继续完全由 `residence_permit` 管理。

V4_008 可重复执行，主初始化和 H2 同步。实现未修改前端，也未改变迁移、注销、审批、流动人口、居住证或导出状态机。
